c++文件读写

发表于 2017-07-21 | 分类于 c++

c++读文件

c++读文件使用ifstream类实例化一个对象，然后ifstream类中重载的>>操作符将硬盘上的文本文件读取到内存中，进行操作。需要注意的是，使用ifstream打开一个文件进行读操作时，如果该文件不存在，不会创建新的文件，其默认的文件打开方式为ios::in。

对文件进行读操作后记得调用close函数关闭打开的文件。

c++写文件

c++读文件使用ofstream类实例化一个对象，然后ofstream类中重载的<<操作符将程序中的变量写入存储在硬盘上的文件中，进行保存、Log等操作。需要注意的是，使用ofstream打开一个文件进行读操作时，如果该文件不存在，会创建新的文件，其默认的文件打开方式为ios::out。

同样，对文件进行写操作后记得调用close函数关闭打开的文件。

c++读写文件

对文件读和写的操作，也可以使用fstream类的实例化对象实现，只需要在第二个参数设定文件打开的方式即可进行读操作、写操作、读写操作。文件的读写方式有以下几种：
|:——–:|:———:|:——-:|
|ios::in|打开文件进行读操作，文件不存在不会创建新文件|
|ios::out|打开文件进行写操作，文件不存在则会创建新文件，存在则清空其内容。|
|ios::app|供写，文件不存在则创建，若文件已存在则在原文件内容后写入新的内容|
|ios::ate|文件打开时，文件指针在最后，与ios::in或ios::out结合使用|
|ios::trunc|读写文件前将文件清空，将文件内容清空。|
|ios::binary|以二进制的形式读写文件|
|ios::nocreate|文件不存在不会创建新文件，文件不存在则打开失败|
|ios::noreplace|不覆盖新文件，文件若存在则打开失败|
ofstream以及ifstream对象同样可以设定以上参数。并且可以通过“或”运算设置两个及以上的打开方式。
第三个参数表示的是打开文件的属性：
|0|普通文件|
|1|只读文件|
|2|隐藏文件|
|3|系统文件|
